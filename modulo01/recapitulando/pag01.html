IMAGEM

  • Formatos de imagem

    • Na hora de escolher o formato de imagem, opte sempre pelo formato JPEG(JPG) com uma compactação enre 30% a 50%. Já o PNG, só usamos quando precisamos de transparência

  • Tamanho de imagem

    • Não salve arquivos gigantes e fique diminuindo o tamanho com código. Seu navegador vai levar um tempo para carregar o arquivo do seu servidor e vai exibir a imagem minúscula na tela.

Na segunda, vamos basicamente descrever o contéudo da imagem para acessibilidade de pessoas cegas e também para que o navegador ofereça sua imagem quando buscarem por algo parecido com sua descrição. IMAGEM DINÂMICA

Para criar um site, é preciso adaptar o conteúdo para diferentes tpos de telas(cellular, tablet, tv etc.). Poderíamos editar diretamente no código, mas isto não diminuiria os bytes, e aumentaria o consumo deixando o site pesado.

Então, de forma mais leve, editaremos 3 imagens iguais(ou mais), porém de dimensões diferentes. Uma tamanho G com 1000px, uma tamanho M com 700px e uma tamanho P com 300px.

Abrimos a tag <picture> e, dentro dela, colocamos normalmente a tag <img> com a maior foto. Exemplo:

        
        
            <picture>
                <img src="" alt="">
            </picture>

Agora, em cima da tag img, abrimos a tag <source media=" (max-width: px)" srcset="" type="">, onde dentro do max-width: colocaremos o tamanho da maior foto + uns 50/60 a mais, dentro do srcset colocaremos o arquivo de uma foto menor que essa, ou seja, a foto média, e dentro do type colocaremos o media type da imagem que usamos. Neste caso, jpg.

Ainda, em cima desta ultima tag, podemos adicionar mais um source, porém desta vez adicionando um arquivo ainda menor e o número dentro do width passar a ser o tamanho da segunda imagem + uns 50/60. Exemplo:

    
        
            <picture>
                <source media="(max-width: 750px)" srcset="fotopequena.jpg" type="image/jpeg">
                <source media="(max-width: 1050px)" srcset="fotomedia.jpg" type="image/jpeg">
                <img src="fotograde.jpg" alt"(descrição)">
            <picture>

LISTA

  • Ordenadas: são as listas numeradas, seja com números, números romanos. Usa-se a tag <ol>, e, dentro, cada item deve ficar entre a tag <li>.

    • Para escolher se a lista vai ser listada com números ou letras, coloque type="" após 'ol' e indique se será a, A, i, I ou 1

    Você também pode indicar em qual número ela vai começar, adicionando um start=""(OBS: o start sempre será indicado por números, mesmo que a lista esteja em type=A)

    Exemplo:

    1. Pão
    2. Manteiga
    3. Arroz

  • Não ordenadas: são as listas não numeradas. São indicadas com símbolos. Para escolher um simbolo, basta adicionar type="" dentro da tag e escrever uma das opções: disc(bolinha preta), circle(circulo), square(quadrado)

    • Exemplo:

    • Mario Bros
    • The last of us
    • Beyond two souls

LINK

Toda vez que você está acessando um site e clica em um local para ir para outra página, outro site ou baixar um arquivo, você está interagindo com um hyperlink

Para criar um hyperlink, iniciamos abrindo a tag <a href=""></a>
Dentro das aspas iremos colocar o link da página/site, ou o arquivo de uma página nossa, que desejamos abrir quando clicar este local. E entre os <> colocamos o texto(ou imagem) que deverá ser clicado.

  • Target

    • Para fcar mais prático para o Google, devemos adicionar o target="" que é um atributo que dirá aonde este link será aberto. Dentro das aspas, temos algmas opções para adicionar:
    _blank: Faz o link abrir em outra aba
    _self: Fa o link abrir nessa mesma aba.

  • Rel

    • Depois de target, adicionamos rel="" e dentro das aspas podemos colocar:
    next Se for uma próxima parte do documento atual.
    external Se for para outro site
    prev Se for voltar pra página anterior
    author Se o link for para um site do autor do artigo atual

AUDIO

Para adicionar um áudio, basa abrir a tag <audio preload="">

O atributo preload indica se o áudio será pre carregado ou não e aceita três valores:

  • metadata Carrega apenas as informações sobre o arquivo
  • none Não carrega nada até o usuário apertar o botão de play
  • auto Vai carregar o arquivo inteiro assim que a página for carregada, mesmo que o usuário não aperte o play.

Podemos, depois do preload, adicionar o atributo controls, que seria o player. Caso não seja adicionado, o usuário não pode interagir com ele.

Também podemos adicionar o autplay que inicia o áudio assim que a página for carregada.

Também temos o atributo loop que mantém o áudio tocando repetidamente.

Após enfeitar a tag, devemos, entre ela, abrir a tag <source src="" type="">
Nas aspas de src colocamos o nome do arquivo.
Nas aspas de type colocamos a media type

Diferente da tag picture, vai fncionar de cima para baixo tentando o próximo recurso se o atual não funcionar.

    • Dentro da tag, podemos utilizar atributos iguais os do audio, como o autoplay e o controls(muito importante para o usuário poder interagir com o vídeo.)
    • Dentro da tag podemos escolher o tamanho com o atributo width e escolher uma tumb com o atributo poster
    • Com a tag aberta, colocamos <source src="" type="">
      Nas aspas do src adicionamos o arquivo.
      Nas aspas do type adicionamos o media type.
    • Podemos colocar vários source como na imagem e no audio. ASsim o navegador tem vários possibilidades de carregar o vídeo.

CSS ESTILOS

  1. Inline
    • Esta é a maneira meno prática porém serve para ocasiões especificas. Você pode ir de linha em linha modificando o estilo. Basta abrir, dentro de alguma tag, o <style>. Dentro dele, você pode modificar posição. tamanho, cor, marca-texto, cor de fundo etc.
  2. Internal
    • Para ficar mais organizado e um pouco mais prático, caso queira mudanças em toda a página, abriremos a tag </style> dentro da head e colocamos tudo o que queremos modificat.
      Exemplo: caso queira modificar o fundo para azul, coloque body {blackground-color: blue;}.
      Caso queira modificar todos os h1, ponha h1 {color: blue;}
  3. Alguns códigos
    • Mudar fonte: style="font-family: (nome da fonte);"
    • Mudar tamanho da fonte: style="font-size: (tamanho)px"
    • Mudar cor de um titulo ou paragrafo: style="color: (cor desejada);"
    • Deixar as letras mais espaçadas: style="text-align: justify"
    • Mudar a direção do título/parágrafo: style="text-align: (center, right ou left);"
